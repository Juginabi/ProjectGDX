Welcome to ProjectGDX web pages!

ProjectGDX is a codename for my new tower defence game made using LibGDX cross-platform game development tool. Target of this game is to replicate the action packed gameplay mechanics known from Warcraft3 custom maps of various types. Follow this page for more information and if you are interested of bringing your own ideas to this game engine, feel free to fork the code and make pull request. If you want to speed up the morale and motivation of the author, check out bottom of this page for donation information.

Old'skool entertainment!

Check out some early 0.1 alpha screenshots of the game engine. Engine uses LibGDX tilemap with 2D graphics and engine can support thousands of moving game entities on screen running on Android 4.0+ smartphone.

Next generation action!

ProjectGDX has been built diversity in mind! Maps can be viewed from top down or from the side which brings variations to the play style and map design! With nearly unlimited amount of deadly traps and weapons (currently two [2] implemented lol) you can start to devastate incoming monsters! Check out cool video from the alpha build below!
